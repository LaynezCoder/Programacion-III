# README - Estructuras de Datos (definiciones simples)

## Colas (Queue)
Una **cola** es como una fila en la tienda o en el banco:
- **El primero que entra, es el primero que sale** (FIFO).
- Se usa cuando la gente/acciones van esperando turno.

Ejemplo rápido: fila de pedidos en una panadería.

---

## Pilas (Stack)
Una **pila** es como una torre de platos:
- **El último que entra, es el primero que sale** (LIFO).
- Sirve cuando lo más reciente es lo que se atiende primero.

Ejemplo rápido: botón “deshacer” (undo) en un programa.

---

## Listas (List)
Una **lista** es una colección de cosas en orden:
- Guardas datos uno tras otro.
- Puedes recorrerla, agregar, quitar, buscar.
- Puede ser como una lista de compras o lista de pedidos.

Ejemplo rápido: lista de productos vendidos en el día.

---

## Árboles (Tree)
Un **árbol** es una estructura como “familia” o “carpetas”:
- Tiene un **inicio (raíz)** y luego se divide en **ramas (hijos)**.
- Sirve para organizar cosas por niveles.

Ejemplo rápido: menú de un sistema (opciones y subopciones).

---

## Grafos (Graph)
Un **grafo** es como un mapa de conexiones:
- Son **nodos** (puntos) conectados por **aristas** (caminos).
- Sirve para representar redes, rutas, relaciones, etc.

Ejemplo rápido: calles entre lugares, o amigos en una red social.
